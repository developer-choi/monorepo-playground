# @monorepo-playground/design-system

이 저장소는 디자인 시스템의 모범 사례(Best Practices)를 구현하기 위한 프로젝트입니다.

화려한 디자인보다는 컴포넌트의 핵심 기능과 웹 접근성 준수에 집중하기 위해 **스타일링은 최소화했습니다.**

다만, 프로젝트를 확인하시는 분들의 가시성과 사용자 경험을 고려하여, **스토리북 에서는 별도의 스타일을 적용**했습니다. 

https://design-system-eta-six.vercel.app/

## 실행방법

```bash
# Storybook 실행
npm run storybook

# 빌드
npm run build
```
